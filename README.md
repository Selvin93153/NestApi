# SELVIN API

1. Clonar proyecto
2. Instalar las dependencias
   `yarn install`
3. Clonar el archivo `.env.template` y renombrarlo a `.env`
4. Cambiar las variables de entorno
5. Levantar la base de datos
   `docker-compose up -d`

6. Ejecutar en modo de desarrollo:
   `yarn start:dev`
   รณ
   `npm run start:dev`
